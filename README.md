# BookList App

BookList App es una aplicación web desarrollada con Vue.js que permite visualizar, gestionar y revisar información de una lista de libros. Este proyecto fue creado como parte del proceso de aprendizaje en desarrollo Front-End, aplicando componentes, rutas, manejo de datos y mejoras visuales en la interfaz.

## Objetivo del proyecto

El objetivo principal fue construir una aplicación funcional que permitiera organizar una lista de libros de manera clara, simple e interactiva, aplicando buenas prácticas de desarrollo web y fortaleciendo conocimientos en Vue.js.

## Funcionalidades principales

- Visualización de una lista de libros.
- Acceso al detalle de cada libro.
- Eliminación de libros desde la interfaz.
- Navegación entre vistas.
- Interfaz visual mejorada y responsive.
- Organización del código por componentes.
